Как пользоваться системой аутентификации

Регистрация

Для использования всех возможностей сайта, вам необходимо зарегистрироваться:

  1. Нажмите на ссылку Регистрация.
  2. Заполните форму регистрации, указав:
    • Имя пользователя (логин)
    • Адрес электронной почты
    • Пароль (минимум 8 символов)
    • Подтверждение пароля
  3. Нажмите кнопку "Register" для завершения регистрации.
  4. После успешной регистрации вы будете перенаправлены на страницу входа.

Вход в систему

Для входа в систему:

  1. Нажмите на ссылку Вход.
  2. Введите ваше имя пользователя и пароль.
  3. При желании отметьте опцию "Запомнить меня".
  4. Нажмите кнопку "Log In".

Восстановление пароля

Если вы забыли пароль, вы можете восстановить доступ к своей учетной записи:

  1. Нажмите на ссылку Восстановить пароль.
  2. Введите адрес электронной почты, указанный при регистрации.
  3. Нажмите кнопку "Request Password Reset".
  4. На указанный адрес будет отправлена ссылка для сброса пароля.
  5. Перейдите по ссылке из письма.
  6. Введите новый пароль и его подтверждение.
  7. Нажмите кнопку "Reset Password".
  8. После успешного сброса пароля вы сможете войти в систему с новым паролем.

Выход из системы

Для выхода из системы:

  1. Нажмите на ссылку "Выход" в верхней части страницы.
  2. Вы будете автоматически выведены из системы и перенаправлены на главную страницу.
